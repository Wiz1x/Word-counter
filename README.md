## Название

Word Counter

## Описание

Простое расширение для Visual Studio Code, которое считает количество слов в открытом файле. 

- Показывает живой счётчик в статус-баре: общее количество слов, а при выделении текста — формат `выделение/всего` (например, `3/120 слов`).
- Команда `Count Words` (вкладка Command Palette) показывает информационное сообщение с количеством слов во всём файле.

## Функционал

- Подсчёт слов выполняется по любым пробельным символам (пробел, таб, перевод строки).
- Пунктуация остаётся частью слова (т.е. `hello,world` считается одним токеном).

## Установка и запуск в VS Code (macOS)

1. Убедитесь, что у вас установлен Node.js и npm:

```bash
node -v
npm -v
```

Если команды нет, установите Node.js (варианты):

- Через Homebrew:

```bash
brew install node
```

- Или через nvm (рекомендуется для управления версиями):

```bash
curl -fsSL https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.4/install.sh | bash
# затем перезапустите терминал и:
nvm install --lts
nvm use --lts
```

2. В корне проекта установите зависимости и скомпилируйте TypeScript:

```bash
cd /Users/wiz1x/Documents/Sdt/Lab3/word-counter
npm install
npm run compile
```

3. Откройте проект в VS Code:

```bash
code /Users/wiz1x/Documents/Sdt/Lab3/word-counter
```

4. Нажмите F5 (Run → Start Debugging). Откроется новое окно — Extension Development Host. В нём откройте любой текстовый файл — внизу вы увидите счётчик слов.

5. Вызов команды вручную: Cmd+Shift+P → введите `Count Words` → нажмите Enter. Появится информационное сообщение вида:

```
В файле 120 слов
```

## Замечания

- Алгоритм подсчёта простой и подходит для большинства латинских языков; для корректного подсчёта слов в CJK (китайский, японский, корейский) нужен иной подход.
